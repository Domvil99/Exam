Assignment name  : do_op
Expected files   : do_op.c
Allowed functions: atoi, printf, write
--------------------------------------------------------------------------------

Write a program that takes three strings:
- The first and the third one are representations of base-10 signed integers
  that fit in an int.
- The second one is an arithmetic operator chosen from: + - * / %

The program must display the result of the requested arithmetic operation,
followed by a newline. If the number of parameters is not 3, the program
just displays a newline.

You can assume the string have no mistakes or extraneous characters. Negative
numbers, in input or output, will have one and only one leading '-'. The
result of the operation fits in an int.

Examples:

$> ./a.out "123" "*" 456 | cat -e
56088$
$> ./a.out "9828" "/" 234 | cat -e
42$
$> ./a.out "1" "+" "-43" | cat -e
-42$
$> ./a.out | cat -e
$


Escribe un programa que tome tres cadenas:

La primera y la tercera son representaciones de enteros con signo en base 10
que caben en un int.

La segunda es un operador aritmético elegido entre: + - * / %

El programa debe mostrar el resultado de la operación aritmética solicitada,
seguido de un salto de línea.
Si el número de parámetros no es 3, el programa simplemente muestra un salto de línea.

Puedes asumir que las cadenas no tienen errores ni caracteres extraños.
Los números negativos, tanto en entrada como en salida, tendrán uno y solo un - inicial.
El resultado de la operación cabe en un int.
